# 交通標誌設計規範研究筆記

## 核心設計原則

### 1. 顏色編碼系統
根據美國交通部(DOT)與 MUTCD 標準:

| 顏色 | 意義 | 用途 |
|------|------|------|
| **紅色** | 停止/禁止 | 停止標誌、讓行標誌、禁止標誌 |
| **黃色** | 警告/注意 | 一般警告訊息 |
| **綠色** | 許可/指引 | 允許的車輛移動或方向指引 |
| **藍色** | 服務/資訊 | 道路使用者服務、旅遊資訊、疏散路線 |
| **橙色** | 施工/警告 | 道路施工區警告與指引 |
| **白色** | 規範 | 規範標誌(如速限) |
| **棕色** | 文化/休閒 | 公共休閒或文化景點指引 |
| **螢光黃/綠** | 行人/學校 | 行人穿越道與學校區域 |

### 2. 形狀編碼系統

| 形狀 | 意義 |
|------|------|
| **八角形** | 停止 |
| **倒三角形** | 讓行 |
| **菱形** | 警告前方可能危險 |
| **五邊形** | 學校區域 |
| **圓形** | 鐵路平交道 |
| **矩形(直立)** | 規範標誌 |
| **矩形(橫向)** | 指引標誌 |

### 3. 符號設計原則

**為什麼使用符號而非文字?**
- 更快速被大腦識別
- 跨越語言障礙
- 適應視覺障礙
- 高速行駛時更易辨識
- 國際通用

**符號設計要求:**
- 簡潔清晰
- 高對比度
- 易於辨識
- 一致性
- 夜間可見(反光材質)

## 應用到 AI 任務管理系統的設計策略

### 顏色分級系統

**一級 | 營收**(直接賺錢)
- **主色:** 金黃/橙色系(象徵金錢、價值)
- **底色:** 深橙色或金色漸層
- **圖案:** 白色或黑色高對比

**二級 | 流量**(增加流量)
- **主色:** 藍色系(象徵流動、成長)
- **底色:** 深藍色或藍綠漸層
- **圖案:** 白色高對比

**三級 | 行政**(流程優化)
- **主色:** 灰色/銀色系(象徵穩定、規範)
- **底色:** 深灰色或銀灰漸層
- **圖案:** 白色或黑色高對比

### 形狀分級系統

**一級 | 營收**
- **基礎形狀:** 圓形(象徵完整、目標)
- **變化:** 圓形內加箭頭、金幣、圖表等

**二級 | 流量**
- **基礎形狀:** 菱形或箭頭(象徵方向、成長)
- **變化:** 向上箭頭、波浪、曲線等

**三級 | 行政**
- **基礎形狀:** 方形或矩形(象徵規範、穩定)
- **變化:** 方形內加勾選、文件、齒輪等
