# AI 任務管理系統 - 符號系統交付文件

## 📦 交付內容總覽

本次交付包含完整的**交通標誌風格任務符號系統**,共30個高品質符號圖形,以及配套的 AI 自動判別機制。

## ✅ 已完成項目

### 1. 符號系統設計

**設計原則**:
- ✅ 參考交通標誌設計規範(顏色、形狀、符號)
- ✅ 高對比度設計(白色圖案 + 彩色底)
- ✅ 三級分類系統(營收/流量/行政)
- ✅ 形狀編碼(圓形/菱形/方形)
- ✅ 顏色編碼(金色/藍色/灰色)

### 2. 符號圖形生成

**一級 | 營收符號(10個)**:
- 直接銷售、客戶拜訪、簽約成交、產品展示、報價提案
- 訂單處理、收款催款、合約審核、定價策略、營收分析

**二級 | 流量符號(10個)**:
- SEO 優化、社群經營、內容撰寫、廣告投放、數據分析
- 網站優化、電郵行銷、影片製作、活動企劃、流量監控

**三級 | 行政符號(10個)**:
- 文件審核、週報月報、流程優化、會議安排、資料整理
- 系統維護、人事管理、預算編列、合規檢查、文書處理

### 3. AI 自動判別機制

**核心函數**:
- `matchTaskSymbol(taskDescription)`: 根據任務描述自動匹配符號
- `getSymbolsByCategory(category)`: 獲取特定分級的所有符號
- `getSymbolById(id)`: 根據 ID 獲取符號

**判別規則**:
- 基於關鍵字匹配(每個符號定義5個關鍵字)
- 自動分級為一級|營收、二級|流量、三級|行政
- 完全由 AI 控制,使用者不可手動修改

### 4. 技術文件

**檔案結構**:
```
client/public/symbols/
├── revenue/          # 一級營收符號(10個 PNG)
├── traffic/          # 二級流量符號(10個 PNG)
└── admin/            # 三級行政符號(10個 PNG)

client/src/data/
└── taskSymbols.ts    # 符號字典檔 + AI 判別函數
```

## 📊 技術規格

### 符號圖形規格

- **格式**: PNG
- **尺寸**: 正方形(實際尺寸由 AI 生成決定)
- **背景**: 彩色底(金色/藍色/灰色)
- **圖案**: 白色
- **風格**: 扁平化、高對比度、類交通標誌

### 符號字典檔結構

```typescript
interface TaskSymbol {
  id: string;              // 唯一識別碼
  name: string;            // 符號名稱
  category: string;        // 分類(revenue/traffic/admin)
  description: string;     // 描述
  keywords: string[];      // 關鍵字陣列(用於 AI 匹配)
  iconPath: string;        // 圖形路徑
  color: string;           // 顏色代碼
  shape: string;           // 形狀(circle/diamond/square)
}
```
