## 七、已知限制與未來擴展

### 7.1 目前為前端模擬
- 所有資料存儲在前端狀態(useState)
- 重新整理頁面後資料會重置
- **建議:**整合後端資料庫以持久化資料

### 7.2 文件與錄音上傳
- 目前僅有按鈕,實際上傳功能需整合檔案處理 API
- **建議:**整合檔案上傳服務(如 S3)

### 7.3 AI 分析
- 目前為模擬分析結果
- **建議:**整合真實 AI API(如 OpenAI GPT-4)進行任務分析

### 7.4 時間追蹤
- 開始/結束時間已記錄,但未計算實際耗時
- **建議:**新增計時器功能,計算並顯示實際投入時間

### 7.5 歷史記錄
- 已完成任務目前直接從列表移除
- **建議:**新增「歷史記錄」頁面,供查詢和匯出
